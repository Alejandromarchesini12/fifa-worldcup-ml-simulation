# FIFA World Cup Outcome Prediction & Simulation

## Overview
This project uses machine learning to predict FIFA World Cup match outcomes and simulate 10,000 full tournaments using model-generated probabilities.

## Objectives
- Clean and merge historical match and FIFA ranking data
- Engineer predictive features (ranking difference, recent form, goal trends)
- Train and compare Logistic Regression, Random Forest, and XGBoost models
- Simulate 10,000 World Cups using predicted probabilities
- Analyze feature importance and model limitations

## Project Structure
data/  
├── raw/ (original datasets)  
├── processed/ (cleaned data)  

notebooks/ (EDA, modeling, simulation)

Python, pandas, NumPy, scikit-learn, XGBoost, Matplotlib, Jupyter Notebook
